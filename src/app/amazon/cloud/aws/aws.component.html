Amazon AWS or Amazon Simple Storage Service is a service offered by Amazon Web Services that provides object storage through a web service interface. Amazon AWS uses the same scalable storage infrastructure that Amazon.com uses to run its global e-commerce network.

Amazon AWS can be employed to store any type of object which allows for uses like storage for Internet applications, backup and recovery, disaster recovery, data archives, data lakes for analytics, and hybrid cloud storage. In its service-level agreement, Amazon AWS guarantees 99.9% monthly uptime, which works out to less than 43 minutes of downtime per month.

Amazon launched Amazon AWS in the United States on March 14, 2006, then in Europe in November 2007.
